Entity Classifier Classc back to ToC or Class ToC

IRI: http://www.cs.queensu.ca/~imam/ontologies/efbo.owl#EntityClassifier

Authors :
Fahim Imam
has sub-classes
Functional Reasoning Categoryc, System Validation Entityc
Set of defined classes to describe a set of Entities. The instances of the entities can be classified via reasoner. This is an "interface class" of the ontology so that users can classify the entities based on their own definition. Example: Activities performed by a System Agent that required inputs from a user. Example: Set of events that happens before the activity of login button being pressed. Example: Set of functionalities of a login interface. Example: A chain of processes that led to another process.
